Comparação entre Linguagens

Linguagens Opinião
Javascript Ando tendo muito contato com javascript, pois estou em um curso com foco em node.js, o javascript é uma linguagem muito importante no geral por fazer parte da tríade do front-end, mas a nível Back-End, eu não gostei muito, a fraca tipagem da linguagem meio que gera situações bem inusitadas.
Java Ainda não tive contato com java, mas pelo que demonizam, parece ser bem difícil, sobre o que sei dela, sei que a mesma é bem importante e muito utilizada para desenvolvimento mobile.
PHP Também não tive contato com PHP, mas sei que a mesma é muito importante e muito utilizada para desenvolvimento Web, na parte do Back-End.
Python Foi a linguagem no qual comecei no mundo da programação, é uma lingugagem amigável falando em questão de syntaxe, muito útil para os mais diversos tipos de uso e com uma comunidade gigante, você basicamente consegue realizar de tudo com python.
C# Nunca tive contato com C# e não sei para o que é utilizada.
Kotlin Recentemente estou aprendendo Kotlin para usá-lo no projeto de residência, estou gostando bastante, sei que foi uma linguagem criada para armonizar com o java na criação de aplicativos android mas evoluiu ao ponto de se tornar a linguagem mais utilizada para a criação desses aplicativos.
